## SRP 单一职责
> 模块只对某一类行为负责

```js
import React, { useState } from 'react';

// 模块内部可以做很多事，但仅提供有限数据、方法给外部调用
const [count, setCount] = useState(0);

// 外部利用有限内容做自适应扩展
const add5 = () => setCount(count + 5);
const add10 = () => setCount(count + 10);
```
