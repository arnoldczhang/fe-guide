# 人类简史

## 认知革命

- 文化演化（让很多人相信一个故事，从而合作）
- 人类是造成物种属科灭绝的最终祸首

## 农业革命

- 反过来想，是农作物、动物驯化了人类，驯化成功的才存活下来

- DNA的拷贝数多=演化成功

- 演化成功≠个体幸福

- 游牧 -> 定居 -> 活动范围减小
