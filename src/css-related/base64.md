# base64

## 字符
- 组成
  *26个大小写英文字母
  * 10个数字
  * / +
- 6bit字节码

## ASCII
- 范围是0-127
- 0-31是控制字符，32-126是可打印字符
- 8bit字节码

## 本质
用6bit字符来表达原来8bit字符
