## 组成部分

### Custom elements

> 允许自定义元素标签及其自定义行为

### Shadow Dom

> 允许封装“影子”dom树，挂到到元素下，这样能保证元素功能私有，样式也不会和其他地方冲突

### HTML templates

> 可以使用<template />和<slot />编写不显示在页面中的标记模板，而且做到了可复用
