- 虚拟滚动：virtual-list

## 性能指标

| 指标  | 含义                    | 区别                |
| --- | --------------------- | ----------------- |
| FP  | 首次渲染（可能空白）            | 无内容要求             |
| FCP | 首次实际内容渲染              | 必须包含图片、文本         |
| FMP | 首次有效渲染（已弃用）           | 太主观，用`LCP`替代      |
| LCP | 最大内容元素渲染              | 核心指标              |
| FID | 首次输入延迟，仅在用户交互时触发      | 100毫秒内算好，长任务会导致恶化 |
| CLS | 累积布局偏移，统计页面加载中的意外布局偏移 | 0.1以下算好，建议固定元素宽高  |

### 使用建议

- 加载性能：优先关注LCP和FCP，这是现代性能监控的核心指标（Google Web Vital标准）

- 交互性能：关注FID

- 页面稳定性：关注CLS
